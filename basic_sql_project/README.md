# Basic SQL: Venture Funds & Investments

Project brief (PDF): [basic_sql_project.pdf](basic_sql_project.pdf)

## Overview
This project is an SQL case study based on a relational database about venture funding and company outcomes.  
The database includes companies, venture funds, funding rounds, investments, acquisitions, and people/education tables.

## Objectives
- Explore dataset coverage (statuses, geography, funding totals).
- Analyse funding activity by country/category and over time.
- Segment funds by investment activity and compare behaviour across groups.
- Analyse acquisitions and compare deal values to prior investment.
- Use people/education tables only in aggregated form.
